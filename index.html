Cómo activar el GPS

El navegador bloquea la ubicación si la app se abre como archivo (file:// o content://, como desde el explorador). Necesita abrirse desde un enlace seguro. Opciones, de la más fácil a la más técnica:

1. Subirla a un hosting gratis (recomendado). Arrastra este archivo a app.netlify.com/drop o súbelo a GitHub Pages. Te da un enlace https:// que abres en Chrome y el GPS funciona. Ideal para el campo.

2. Servidor local en el teléfono. Con Termux: python3 -m http.server 8000 en la carpeta del archivo y abre http://localhost:8000. localhost cuenta como seguro, así que el GPS funciona 100% sin internet.

3. En PC. python -m http.server y ábrela en http://localhost:8000.
